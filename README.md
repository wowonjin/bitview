# Bitview 빗뷰 - 비트코인 이더리움 리플 솔라나 실시간 시세 | 바이낸스 바이비트 펀딩비 계산기 | 암호화폐 백테스트 트레이딩

**Bitview (빗뷰)**는 비트코인, 이더리움, 리플, 솔라나 등 주요 암호화폐의 실시간 시세와 전문가급 차트 분석, 백테스트 트레이딩 기능을 제공하는 종합 암호화폐 플랫폼입니다.

🔥 **주요 키워드**: Bitview, 빗뷰, 비트코인, 이더리움, 리플, 솔라나, 트레이딩, 백테스트, 벡테스트, 바이낸스, 바이비트, 펀딩비, 펀딩비계산기, 복리계산기, 암호화폐, 실시간 시세, 차트 분석, 환율, 나스닥

## 🚀 핵심 서비스

### 📊 실시간 암호화폐 시세 모니터링
- **비트코인 (Bitcoin)** 실시간 가격 추적
- **이더리움 (Ethereum)** 시세 분석
- **리플 (XRP)** 가격 변동 모니터링  
- **솔라나 (Solana)** 실시간 차트
- 200+ 주요 암호화폐 실시간 데이터
- 생태계별 필터링 (비트코인, 이더리움, 바이낸스, 솔라나 등)
- 즐겨찾기 및 관심종목 관리

### 💰 바이낸스 바이비트 펀딩비 계산기
- **바이낸스 (Binance)** 선물 펀딩비 실시간 계산
- **바이비트 (Bybit)** 펀딩비 계산기
- 레버리지별 펀딩비 수익 시뮬레이션
- 일간/월간/연간 펀딩비 수익 예측
- 실시간 펀딩비율 API 연동

### 📈 암호화폐 복리계산기
- 암호화폐 투자 복리 수익 계산
- 바이낸스/바이비트 수수료 포함 계산
- 레버리지 거래 수익률 시뮬레이션
- 복리 횟수별 최종 수익 예측
- KRW 원화 기준 계산

### 📈 실시간 차트 분석 및 트레이딩
- TradingView 전문가급 차트 제공
- 비트코인, 이더리움, 알트코인 차트 분석
- 다양한 시간 간격 지원 (1분 ~ 1주)
- 기술적 지표 및 패턴 분석

### 🔄 암호화폐 백테스트 (벡테스트) 트레이딩
전문가급 백테스트 기능으로 과거 데이터 기반 트레이딩 전략 검증:

#### 📊 고급 백테스트 전략
- **트리플 이동평균 전략**: 5일, 20일, 50일 이동평균 기반
- **MACD + RSI 조합 전략**: 기술적 지표 조합 분석
- **볼린저 밴드 평균회귀**: 변동성 기반 매매 전략
- **스토캐스틱 다이버전스**: 가격 패턴 분석

#### 🎮 실시간 백테스트 리플레이
- 실시간 차트 시뮬레이션
- 0.5x ~ 8x 배속 조절 기능
- 매수/매도 신호 실시간 표시
- 수익률 및 통계 실시간 업데이트

#### 📋 기술적 지표 분석
- **이동평균 (SMA/EMA)**: 추세 분석
- **볼린저 밴드**: 변동성 지표
- **MACD**: 모멘텀 분석
- **RSI**: 과매수/과매도 지표
- **스토캐스틱**: 오실레이터 분석

## 🛠️ 기술 스택

- **Frontend**: React 18, Vite, JavaScript
- **Charts**: TradingView Widgets, Recharts, Custom SVG
- **API Integration**: 
  - CoinGecko API (실시간 암호화폐 가격)
  - Binance WebSocket (실시간 시세)
  - Binance Futures API (펀딩비)
- **Styling**: CSS3, Responsive Design
- **Authentication**: React Context API

## 📦 설치 및 실행

### 환경 요구사항
- Node.js 16.0.0 이상
- npm 또는 yarn 패키지 매니저

### 로컬 개발 환경 설정
```bash
# 저장소 클론
git clone https://github.com/your-username/bitview.git
cd bitview

# 의존성 설치
npm install

# 개발 서버 실행 (localhost:3000)
npm run dev
```

### 프로덕션 빌드
```bash
# 프로덕션 빌드
npm run build

# 빌드 결과 미리보기
npm run preview
```
